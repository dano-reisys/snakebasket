<h1>pip<a class="headerlink" href="#pip" title="Permalink to this headline">¶</a></h1>
<p><cite>pip</cite> is a tool for installing and managing Python packages, such as
those found in the <a class="reference external" href="http://pypi.python.org/pypi">Python Package Index</a>. It&#8217;s a replacement for
<a class="reference external" href="http://peak.telecommunity.com/DevCenter/EasyInstall">easy_install</a>.</p>
<div class="highlight-python"><pre>$ pip install simplejson
[... progress report ...]
Successfully installed simplejson</pre>
</div>
<p>Upgrading a package:</p>
<div class="highlight-python"><pre>$ pip install --upgrade simplejson
[... progress report ...]
Successfully installed simplejson</pre>
</div>
<p>Removing a package:</p>
<div class="highlight-python"><pre>$ pip uninstall simplejson
Uninstalling simplejson:
  /home/me/env/lib/python2.7/site-packages/simplejson
  /home/me/env/lib/python2.7/site-packages/simplejson-2.2.1-py2.7.egg-info
Proceed (y/n)? y
  Successfully uninstalled simplejson</pre>
</div>
