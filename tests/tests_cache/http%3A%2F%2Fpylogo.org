<h2>起源と歴史</h2>
<p>

アートメイクは、中国で「眉の薄い女性は幸せになれない」と古来より言い伝えられてきたことが起源となっています。<br>
眉毛の薄い女性達は、墨などで眉毛を濃くすることで運勢を良くしようとし、<br>
そして運勢が二度と落ちないようにと願いを込めて色が落ちない工夫をしたと言われています。<br>
時が流れ、肌を染色するようになったのが現在のアートメイクの始まりのようです。約3000年前のお話になります。<br><br>

■中国から台湾、台湾からアメリカへ<br>
「アートメイク　大阪」メイクの染色技術は台湾に伝わっていきます。<br>
そして台湾に旅行に来ていたアメリカ人の美容科が技術を学習し母国に持ち帰り、
さらなる研究を重ねた結果、現在のアートメイクに近いものが生まれたとされているのです。<br>
さらに天然の色素についても研究を重ね、美容のみならず医療補助技術として利用できるまでに発展しました。<br><br>

■「アートメイク」という言葉は俗語<br>
アートメイクという言葉は、実は日本で生まれた俗語になります。<br>
正式名は「パーマネント　メイクアップ（permanent make up）」といいます。<br>
今から30年～40年前に日本に入ってきたと言われてます。<br><br>

歴史を重ねて技術が向上しているアートメイク　大阪で試してみませんか？<br></p>
